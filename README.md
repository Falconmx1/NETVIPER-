# 🐍 NETVIPER - La serpiente de tus redes

**NETVIPER** es una herramienta de reconocimiento y análisis de red 
diseñada para los que no tienen miedo. 

✅ Multiplataforma (Linux/Windows)
✅ Escaneo de puertos relámpago
✅ Detección de dispositivos ocultos
✅ Monitoreo de tráfico en tiempo real
✅ Modo sigiloso para no dejar huella

No es para novatos. Es para los que cazan redes.

Comandos que tendrá:
netviper --scan 192.168.1.0/24    # Escanea toda la red
netviper --ports 1-1000           # Escaneo de puertos
netviper --stealth                 # Modo sigiloso activado
netviper --sniff                   # Intercepta tráfico
netviper --ping-sweep              # Descubre hosts vivos
